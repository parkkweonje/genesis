# 🏢 제네시스 (Genesis)

AI가 **기획 → 제작 → 다듬기 → 홍보 → 응대**까지 스스로 운영하는 회사 저장소입니다.

## 조직

| 팀 | 역할 | 서브에이전트 |
|----|------|--------------|
| 👑 사장 | 최종 승인·방향 결정 | 사람(나) |
| 🧭 마스터 | 업무 분배·검수 | 메인 Claude Code |
| 📋 기획팀 | 사이트 방향·콘텐츠 기획 | `planning-team` |
| ✍️ 블로그작가 | 블로그 글 작성·발행(매일) | `blog-writer` |
| 💻 개발팀 | 제작·수리 | `dev-team` |
| 🎨 디자인팀 | 화면 다듬기 | `design-team` |
| 📣 마케팅팀 | 홍보·카드뉴스·SEO | `marketing-team` |
| 💬 CS팀 | 손님 문의 응대 | `cs-team` |

팀 정의는 [`.claude/agents/`](.claude/agents/)에, 회사 운영 지침은
[`CLAUDE.md`](CLAUDE.md)에 있습니다.

## 사용법

사장이 마스터(Claude Code)에게 요청하면, 마스터가 알맞은 팀에 분배하고
결과를 검수해 보고합니다. 특정 팀을 직접 부르려면 이렇게 지시하세요.

```
기획팀에게: 수험생 대상 정보 사이트 구성안 3개 뽑아줘
블로그작가에게: "재택 부업 추천 8가지" 주제로 새 글 써서 발행해줘
개발팀에게: 상단 네비게이션 모바일 햄버거 메뉴 붙여줘
디자인팀에게: 메인 히어로 섹션 여백·색 다듬어줘
마케팅팀에게: 인스타 카드뉴스 5장 문구 써줘
CS팀에게: "환불 되나요?" 문의 답변 초안 만들어줘
```

## 개발

빌드가 없는 순수 HTML/CSS/JS 사이트입니다.

```bash
python3 -m http.server 8000
# → http://localhost:8000
```
