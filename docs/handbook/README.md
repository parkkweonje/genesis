회사 업무 매뉴얼(루트 [`CLAUDE.md`](../../CLAUDE.md))을 바탕으로, **각 팀이 함께 작성**했습니다.

## 핵심 요약 (외워둘 것)

- **작업 원칙:** 정확성 > 검증 > 최소 변경 > 명확성 > 유지보수성
- **업무 흐름:** 접수 → 분류 → 분배 → 검수 → 보고 → 배포
- **배포:** `git push origin main` 으로만
- **검증:** 정적 사이트는 저장소 **상위 폴더**에서 `python3 -m http.server` (그래야 `/genesis/` 경로가 산다)
- **접근:** 사장 바탕화면 `개인` 폴더 🚫 금지 / Google Drive·Notion·Calendar 등 ✅ 자유
- **금지:** 검증 없이 "됐다" 말하기, 과장·허위 수익 표현(애드센스 정책), 요청 범위 밖 변경
